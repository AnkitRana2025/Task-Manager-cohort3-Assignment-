# 📝 Task Manager App (Vanilla JavaScript)

A simple yet powerful **Task Manager Web App** built using **HTML, CSS, and Vanilla JavaScript**.  
It includes features like task CRUD operations, filtering, and local storage persistence.

---

## 🚀 Features

- ➕ Add new tasks
- ✏️ Edit existing tasks
- 🗑️ Delete tasks
- ✅ Mark tasks as completed
- 🔍 Filter tasks (All / Completed / Pending)
- 💾 LocalStorage support (data persists on refresh)
- ♻️ Reset all tasks feature
- 🎨 Clean and responsive UI

---

## 🧠 Concepts Used

- DOM Manipulation
- Event Delegation
- Array Methods (map, filter, find)
- LocalStorage API
- Dynamic Rendering
- Form Handling

---


## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (ES6)
